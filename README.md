## 需求描述
###### 1.供能监视
	能源站实时供能数据监视，包括CCHP机组、直燃机组、离心式电制冷机、热水锅炉，
	用电功率、用气流量、用水流量，产电功率，产冷热功率。
###### 2.用能监视
	用户实时用能数据监视，包括用电数据、用水数据、燃气数据、冷热数据。
###### 3.配网监视
	机场配电网数据监视，主要包括110kV升压站数据，10千伏变电站数据。
###### 4.空调数据监视
	机场航站楼空调末端数据监视，包括区域温度，二氧化碳浓度，送风机开度。
###### 5.节能减排
	碳氧化物、氮氧化物减排量展示。
